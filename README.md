## 1. 课程
### 1.1 信息解析
#### 1.1.1 课号
课号由**4**部分组成，我们用CSC110Y1和MAT223H1作为例子：
* 前三个字母表示专业 -> CSC代表computer science，BIO代表biology
* 第一个数字代表课程等级 -> 1代表100level的课，2代表200level的课，**不代表几年级才能上**，只要满足限制要求(1.1.2)就可以
* 最后一个字母代表学分 -> Y代表1.0学分，H代表0.5学分，绝大部分的情况Y是年课，H是学期课，但CSC110Y1是个例外。
* 最后一个数字代表校区 -> 1代表UTSG，3代表UTSC，5代表UTM
#### 1.1.2 限制
**prerequisite** - 在上这门课之前必须已经**拿到学分**的课，有时候还会有分数要求（图中例子）或者学分要求（例 至少有4.0学分）

**corequisite** - 在上这门课的时候**同时在上或者已经拿到学分**的课

**exclusion** - 上了exclusion的课就不能再上这门课 （图中例子：上了CSCB36H3就不能再上CSC240H1）

每门课都有自己的br属性，一共有5大类br。在最后一行能看到这门课的br属性。有些Y课会有两种br属性，一般来说是算每种0.5学分。（图例2，会获得0.5br1和0.5br2）

#### 1.3.1 Priority
* P代表在**Priority Period**(某个日期前)，只有**指定的群体**（包括但不限于某年极，某专业）可以enroll这门课(图例1)

* R1代表**只有**指定群体可以enroll这门课 （图例2）

如果你没有priority，那么最好先调查抢的人多不多并且找好planB。或者enrol一个有priority的open program让自己先有priority，后面再drop掉。

#### 1.3.2 Waitlist

可以把一个课当成饭店，只有有限的座位(availability)，当店里已经坐满后你只能排队(waitlist)。有人吃完了空出了桌子(把课drop了)就会让队伍的第一个人进去吃饭(enrol进了)。
但某个时间后(一般是开学一个周左右)，店家就取消了排队，变成了抢桌子，只要有空桌子你就可以去抢，谁先坐下谁就能吃饭(先到先得)。

一般来说抢课的时候waitlist排名在course size的前15%都不用担心，等一等开学前都能进的，但也要根据课程难度来看，有的绝世大水课很少有人drop，有的课很难开学没几天就很多人drop。

<img width="1256" height="199" alt="image" src="https://github.com/user-attachments/assets/8339b2d1-b21d-4cdc-a9a3-21ba22bac753" />

#### 1.3.3 Course Load
在Acorn的选课页面，可以看到每个学期的current course load。

在Priority Period, 每个学期的maximum是2.5学分。在Priority Period结束后，每个学期的maximum是3学分。

如果想要上更多课，需要找College Registrar申请，一般在大一fall申请会被拒绝，因为学校不知道你水平如何。

需要注意的是，在waitlist里的课也会算作course load的一部分，但同一门课waitlist多个不同session仍然只算一门课的course load。

<img width="612" height="272" alt="image" src="https://github.com/user-attachments/assets/218e68e7-0f1e-4ae8-967f-05fe22aaec79" />

### 1.4 cr

cr就是这门课只算学分不算gpa，只要及格就行。整个本科最多只能cr 2.0学分，在cr ddl前可以随时cr或者取消cr。
一门课cr后只会保留自己的br和等级信息，比如cr NFS284后，它就成了了0.5 br4 200+的学分。
需要注意的是cr后的课不能再作为满足专业完成要求的课。

### 1.5 转学分
大部分情况转学分的结果和cr非常类似，只有学分，br和等级的信息。
某些转学分的课可以被考虑代替进系课或者prerequisite，这种情况最好先联系department确认后续步骤。

## 2. 专业

一个大类的介绍通常会包括它们开设专业的overview，申请方法，和实习项目。

### 2.1 类别
**Specialist** - 要上的课最多，对于深度和广度都有更高的要求

**Major** - 上完能对专业有一定程度的认知，但高年级课程量的要求没有specialist的多，

**Minor** - 上着玩，起到扫盲作用，有可能对高年级课程有数量限制(比如最多上1.0学分的300+)

**Focus** - 选课指南，需要依附于Specialist或者Major，不是必要的。

<img width="560" height="171" alt="image" src="https://github.com/user-attachments/assets/9e1021af-b2fa-4424-b05d-2fec4250ff3e" />

### 2.2 进系
一般分为两种program，open和limited
* open - 你想进就进，没有任何门槛
* limited - 会有详细的进系要求，比如必须上某些课并且至少达到某个分数，一般来说会择优录取，每年分数线都会不同

### 2.3 完成要求
每个专业都会有非常详细的课程要求，大致分为两类**必修**和**自由选择**。
* 必修课一般都会标明课号，比如CSC369
* 自由选择一般是从XXgroup里上XX学分，或者至少XX学分的X00level。
  这时候就体现了focus的用处。自由选择的范围太广了，逐个逐个课看是否符合自己的规划非常麻烦。而focus会把相关的课整合在一起，就像在和你说
  ”你想学xxx领域的东西吗，上这些课吧！“，节省了大量时间。但需要注意的是，focus只是推荐，如果你不想上它推荐的某些课完全没关系，要有自己的判断。

## 3. 毕业要求
最方便理解毕业要求的办法是使用[Degree Explorer](https://degreeexplorer.utoronto.ca)。
### 3.1 学分等级要求
总的来说毕业至少要20.0学分，通常来说一年五个学分四年毕业正好。对于每个等级以上的课都有最少个数要求，比如至少上13.0学分的200 level+(300，400都算200+)，
意味着你不能一直上低level的课刷分。
值得注意的是对于300+的要求不能有超过1.0学分的转学分，相当于你必须在ut本校上至少5.0学分的300+。
<img width="1457" height="486" alt="image" src="https://github.com/user-attachments/assets/ae0a5851-0bc5-48a0-8856-9ad591f5370e" />

### 3.2 br要求
br有关的信息详见[1.1.3](#br)。完成br要求有两种方式：
1. 选择四个类别，每个上至少1.0学分 (1+1+1+1)
2. 其中三个类别上至少1.0学分，剩余两个类别各上0.5 (1+1+1+0.5+0.5) （图中例子）
<img width="1451" height="220" alt="image" src="https://github.com/user-attachments/assets/a567a2aa-5312-47b6-a6a3-a815f32a9cac" />

### 3.3 专业要求
最低限度可接受的combo有：
1. 1个specialist
2. 2个major
3. 1个major + 2个minor

专业上限是3个并且第三个只能是minor，你当然可以选择specialist + major + minor。

注意有最低学分不同要求，最多只有X学分可以算作同时满足多个专业。

### 3.4 其他要求
1. 课号前三个字母相同(同一个专业)的课不能超过15.0学分
2. 如果你的专业是specialist，必须上只要1.0学分的400+
3. major和minor之间必须至少有12.0学分的不同

## 4. 实习/research
### 4.1 ASIP
[ASIP](https://www.artsci.utoronto.ca/current/experiential-learning/internships/asip)是art & science的实习项目，在拥有ASIP资格的专业的学生可以在大一结束后的暑假申请，每年暑假开放申请，最晚大三暑假申请，申请需要看gpa和文书。

进了ASIP后需要上四门培训课，第一门课完成指定作业后可以解锁ASIP的工作平台。ASIP的工作平台现在还是有点烂，大部分人都是在官网投递，只需要ASIP提供的COOP工签。本质上ASIP和PEY，COOP这些没什么区别，只是名字不同。

如果你是国际生并且有实习量的需求，我强烈建议进入ASIP，为了那个工签。没有工签你就只能在暑假实习，竞争非常大。工签允许你在fall/winter实习，并且不会影响以后申请毕业工签。
如果你是本地生，我建议可以先进入ASIP并在找到第一个工作后退出。有些公司会偏向于在internship program的学生，因为它们有funding。

ASIP官网上有很多的work term schedule，但不是必须根据这些来。如果你找到了工作但不符合他们给的schedule，给他们发邮件拿approval就好了。

### 4.2 WorkStudy
俗称打黑工，大部分只能拿最低时薪，但是一个很好的积累经验的机会。每年有两个term，summer和fall&winter。
summer在四月底五月初开始申请，做四个月。fall&winter在八月底开始申请，做八个月。需要在[CLNx](https://clnx.utoronto.ca/home.htm)上申请。

WS的工作非常多元，有各种文员帮忙整理文件，it support，活动策划。最重要的是有很多lab招research assistant，也有招software developer的。

申请一般需要提交resume，cover letter。有些岗位会让你提交录像，作品集，transcript和 Co-Curricular Record。

CCR简单来说就是你对学校做的贡献记录，最简单的获得办法是注册某门课的[RSG](https://sidneysmithcommons.artsci.utoronto.ca/recognized-study-groups/join/)，也就是学习小组。

### 4.3 其他
其他research program还有ROP，UTEA这种，每个department也会有自己的research program，建议去自己的department官网看看。还有一些research课(比如CSC494)或者和业界合作的project课(比如CSC301)都是积累经验的好机会。
